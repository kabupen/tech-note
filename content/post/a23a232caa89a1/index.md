# こまごまとした Tips


統計検定を勉強中に遭遇した「そういえば何だっけこれ」という、細々とした tips を備忘録として残しておきます。


### グラフの概形について

グラフの概形を書く際には増減表を用いて考えていきます。

* １次導関数 $f^\prime (x)$ は傾きを表す
* ２次導関数 $f^{\prime\prime} (x)$ は傾きの増加量を表す

これらの組み合わせで関数がどのように増減しているかが分かります。

| 項目                   |  |                  |               |                     |
|:--:|:--:|:--:|:--:|:--:|
| $f^\prime (x)$         | $+$               | $+$               | $-$   | $-$        |
| $f^{\prime\prime} (x)$ | $+$               | $-$               | $+$   | $-$        |
|                        | 下に凸             | 上に凸             | 下に凸 | 上に凸      |
